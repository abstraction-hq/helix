# Helixpay CLI
CLI Wallet to interact with Helixpay protocol.

## Usage

Install helix globally

```
npm install -g @helixpay-xyz/helix
```

Open helix wallet

```
helix
```

## Running the tests

```
npm run test
```

## Linting

```
npm run lint
```
